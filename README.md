# AdjMatrixGraph (CSCE146)

Java implementation of an adjacency-matrix graph with traversal checker.

## Files
- `AdjMatrixGraph.java` — adjacency matrix implementation with DFS and BFS methods.
- `AdjMatrixChecker.java` — small runner/checker (contains `main`) that demonstrates using the graph.

## How to compile & run (command line)
1. Clone repo or download files.
2. Compile:
   ```bash
   javac AdjMatrixGraph.java AdjMatrixChecker.java
3. Run:
java AdjMatrixChecker
